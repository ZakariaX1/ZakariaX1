## Personal Projects
### 1. [SSyncSpace](https://github.com/ZakariaX1/SSyncSpace)
What started out as a project to help an online discord community to organize their thoughts for the events they host for their community, blew up into a grand idea in my head to expand it before I've even started. Unfortunately, because I'm a bit of a perfectionist I spent a lot of time looking for "the best" frameworks to use and have been worrying too much about the external design, so the project is very much still in it's infant stages where I might still even change what frameworks I'm using, but that's relatively normal for a personal project. The project will have "Spaces" that target different applications, platforms or ideas that then have their own tools. The first Space will be the initial project idea for the Discord Events, where event teams can get their event ideas in one place. ### 2. [MCRunner](https://github.com/ZakariaX1/mc_runner)
Closely related to SSyncSpace, this project is meant to serve as a community event plugin for MineCraft that the community can have fun with and should work together with a bot that has some kind of digital free currency or XP that the audience can use to help or hurt a player (or runner) on a server as the runner tries to complete the game. The audience members can get a reward if the runner wins and has helped the runner by getting a percentage of the prize pool, or a player can gain a big prize if their mob or item to hurt the player actually kills the runner.
The project has the minecraft plugin as well as a dummy discord bot, although the discord bot of Choice will likely be integrated with Sero (more about Sero further down)

Once more, due to lack of time and unfortunately also due to lack of interest of others, this project has been temporarily put on hold.

I've offered to port the application part to also work on Linux, as it'd be nice to have screensavers, pixel shift and now playing media available on Linux.
